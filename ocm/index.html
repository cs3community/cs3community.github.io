Open Cloud Mesh

 

Open Cloud Mesh (OCM) is a server-to-server protocol designed to enable federation between independent Enterprise File Sync and Share (EFSS) platforms. Initially conceived of in 2015 and deployed since 2016, multiple implementations exist today.
